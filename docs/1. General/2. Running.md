# Running:

To run this project you will need to:

## Install dependecies:
Run `yarn` or `npm install`


## Compile:
Run one of the following:
- `tsc`
- `yarn build`
- `npm run build`
- `./node_modules/.bin/tsc`

*(If you use `ts-node` - skip this step. You know what to do)*


## Start:
Run one of the following:
- `yarn start`
- `npm run start`
- `node dist/app.js`
